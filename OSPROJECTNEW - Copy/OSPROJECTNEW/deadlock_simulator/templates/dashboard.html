Help & Guide

What is Banker's Algorithm?

Banker's Algorithm is a deadlock avoidance algorithm. Before allocating resources to a process, it checks if the system would remain in a safe state. If not, it denies the allocation. This prevents deadlock from happening.

What is Deadlock Detection?

Deadlock Detection allows resource allocation freely, then periodically checks if the system is in a deadlocked state. If deadlock is detected, recovery actions (terminate or preempt) are offered to break the deadlock.

Recovery Actions

Terminate: Kill a deadlocked process and release all its resources.

Preempt: Take one resource from a deadlocked process. Less disruptive, may need multiple attempts.
